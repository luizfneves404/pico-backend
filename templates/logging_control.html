Logging Level Information

SQL Logging Levels:
  • DEBUG: Shows all SQL statements with parameters - useful for debugging but very verbose
  • INFO: Shows SQL statements without parameters
  • WARNING: Shows only warnings and errors
  • ERROR: Shows only SQL errors
  • CRITICAL: Shows only critical SQL errors
Notes:
  • • Level changes are applied immediately
  • • Changes persist until application restart
  • • DEBUG level may impact performance in production
  • • All changes are logged to the application logs
