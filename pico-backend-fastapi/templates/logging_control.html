Logging Information

Logging Levels (Python):
  • CRITICAL (50): Serious error, program may be unable to continue
  • ERROR (40): A serious problem has occurred
  • WARNING (30): Something unexpected happened
  • INFO (20): General information about program execution
  • DEBUG (10): Detailed information for debugging
  • NOTSET (0): No level set (inherits from parent)
Common Logger Patterns:
  • sqlalchemy.engine - SQL statements
  • sqlalchemy.pool - Connection pooling
  • sqlalchemy.orm - ORM operations
  • requests.* - HTTP requests
  • urllib3.* - URL library
  • __main__ - Main application
  • app.* - Your application modules
Important Notes:
  • • Changes are applied immediately
  • • Changes persist until application restart
  • • Lower levels include higher levels
  • • Child loggers inherit from parents
  • • Use wildcards (*) for pattern matching
  • • Be careful with DEBUG in production
